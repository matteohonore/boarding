# Onboarding

L’objectif est simple : créer une application en ReactJS pour afficher une liste de produits et mettre en place des filtres pour pouvoir filtrer cette liste.

Pour cela, la première étape consiste à créer un nouveau projet à l’aide de create-react-app. 

Voici la documentation : [https://create-react-app.dev/](https://create-react-app.dev/)   

Bonne lecture 🤓

Une fois que l’application par défaut fonctionne en local il faut récupérer la liste des produits et ça se trouve ici : [https://dummyjson.com/](https://dummyjson.com/) (plus précisément dans la doc c’est les **products**).

Quand la liste est affichée il faut la filtrer : 3 filtres (ou plus) sont à mettre en place et il faut pouvoir trier la liste suivant plusieurs critères (par exemple par prix, ordre alphabétique et la note).

L’utilisation de librairie de composants est autorisée, il en existe plein : 

- Material-UI
- Ant Design (AntD)
- React Bootstrap
- Grommet
- ….

Pour le style, aucun style n’est imposé : be creative ✨

 Une page d’authentification est à ajouter à l’aide de [https://dummyjson.com/docs/auth](https://dummyjson.com/docs/auth) (mais conseil il n’est pas obligatoire de commencer par cette étape car le plus important est l’affichage et la manipulation de la liste des produits).

Le code doit être push régulièrement sur github comme ça il y a un suivi et si jamais ton PC ne s’allume plus tout n’est pas perdu. 😅

Quand toutes les fonctionnalités sont développées : présentation de l’application et test de l’application par un référent (c’est le partiel 🧐).

Petit plus : faire plus que l’énoncé et ajouter des fonctionnalités !

C’est ton projet, si tu as des idées ajoute des fonctionnalités pour améliorer ton application 🚀
